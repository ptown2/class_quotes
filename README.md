class_quotes
============

It's a PHP Class of QUOTES

#### USAGE: ####
`$quotes = new Quotes();`

`$current_quote = $quotes->select_quote();`

#### FUNCTIONS: ####
`$quotes->set_category('default');`

`$quotes->add_quote('message', 'author');`

`$quotes->select_quote();`

`$quotes->count_quotes();`
